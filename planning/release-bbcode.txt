[b]stolmine firmware v0.7.0-stolmine.9.0.0 + habitat v2.0.0[/b]

10 new units, 67 commits. Highlights:

[b]Impasto[/b] — 3-band multiband compressor. LR4 crossover, per-band FFT spectrum with gain reduction contour, sidechain split through crossover for frequency-aware detection. SSL G-Bus style speed control. Based on tomf's CPR algorithm.

[b]Parfait[/b] — 3-band multiband saturator. 7 shapers per band (tube/diode/fold/half-rect/crush/sine/fractal), SVF morph filter, single-knob compressor, per-band FFT spectrum. 13% idle CPU on hardware.

[b]Rauschen[/b] — 11-algorithm noise and chaos generator. White, Pink, Dust, Particle, Crackle, Logistic, Henon, Clocked, Velvet, Gendy, Lorenz. X/Y controls per algorithm, post-generator SVF morph filter with V/Oct, 3D rotating phase space visualization.

[b]Pecto[/b] — Comb resonator. 16 tap patterns, 4 resonator types (raw/guitar/clarinet/sitar), gate-triggered xform randomization.

[b]Etcher[/b] — Now with 8 depth-controlled transforms (random/rotate/invert/reverse/smooth/quantize/spread/fold) and CV gate trigger.

Also: [b]Varishape Osc[/b] (POLYBLEP), [b]Transport[/b] (gated clock), [b]Spectrogram[/b] (inline FFT analyzer), plus three experimental units in the new catchall package ([b]Sfera[/b], [b]Lambda[/b], [b]Flakes[/b]).

Firmware adds Doom screensaver, threshold labels API, slot machine text input (WIP), maze screensaver, quicksave confirmation guard, favorites scroll fixes.

Requires er-301-stolmine firmware. All packages compatible with vanilla community packages (Accents, sloop, lojik, strike, polygon).
