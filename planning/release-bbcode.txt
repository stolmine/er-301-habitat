[b]stolmine firmware + habitat v2.1.0[/b]

Two new units in the spreadsheet package.

[b]Larets[/b] -- stepped multi-effect processor. Clock-driven 16-step sequencer where each step applies one of 10 audio effects to the incoming signal: stutter, reverse, bitcrush, downsample, filter, pitch shift, distortion, shuffle, delay, comb. Per-step param and tick count, global skew for rhythmic variation, CPR single-band compressor, bipolar global param offset for sweeping all step values together, loop length down to 1 for momentary single-step hold, transform gate for CV-driven randomization. Clock-locked buffer effects (stutter is true beat repeat, shuffle picks fresh slices per loop pass) and Dattorro-style grain pitch shift. Custom overview viz gives per-effect visual feedback (concentric rings for comb, sweeping highlight for reverse, particle fill for bitcrush, etc.) with auto-normalizing Y axis.

[b]Helicase[/b] -- 2-op FM oscillator with discontinuity waveshaping and phase-latched sync. Carrier + modulator each offer 8 OPL3-inspired waveforms. 16 post-FM fold shapes (tanh through Chebyshev T3), modulator feedback with tanh soft clip, lin/expo FM toggle, lo-fi/hi-fi quality option. JF-inspired phase-receptivity sync: pending edges fire only when the modulator phase crosses a programmable threshold (0 = hard sync, 0.5 = soft sync, 1.0 = subharmonic lock), threshold CV-modulatable via sync-ply expansion fader. Three custom visualizations: 3D k-means phase-space metaballs with Voronoi edges, live transfer curve showing the active fold, circular modulator ribbon with Catmull-Rom interpolation.

Only spreadsheet bumped (2.0.0 -> 2.1.0). Other packages unchanged.

Delete the installed spreadsheet lib from the rear SD card to force reinstall:
[code]
sudo rm -rf /mnt/v0.7/libs/spreadsheet/
[/code]

Requires er-301-stolmine firmware. Compatible with vanilla community packages (Accents, sloop, lojik, strike, polygon).
