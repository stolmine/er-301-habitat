[b]er-301-habitat v2.3.0[/b] -- 2026-04-17
Requires firmware: v0.7.0-txo (er-301-stolmine)
Packages: spreadsheet 2.2.0 -> 2.3.1, biome 2.0.0 -> 2.1.0

[b]Highlights[/b]
Two new units -- [b]Colmatage[/b] (breakbeat cutter) and [b]Blanda[/b] (three-input scan mixer). Audio-quality improvements on Helicase and Petrichor. Pecto gets a big CPU reduction via NEON.

[b]New unit: Colmatage[/b]
Clock-driven breakbeat cutter (spreadsheet pkg). Lineage: Nick Collins' BBCut -> Remy Muller's Livecut, refined into a single WarpCut-derived engine. Records to a circular buffer and slices/rearranges it using geometric repeat series -- accelerating repeats bounce, decelerating repeats settle.
[list]
[*][b]clock[/b] -- external clock + subdivision, genuine phrase reset, CV subdiv on expansion
[*][b]block[/b] -- block size weight, BSP mosaic viz with Perlin cloud modulation. Shift: phrase min / phrase max / block max
[*][b]density[/b] -- cut probability vs clean passthrough
[*][b]repeats[/b] -- target repeats/block (2-64). Shift: ritard bias / blend / accel
[*][b]texture[/b] -- bipolar duty (-1 reverse, 0 gate, +1 forward). Shift: amp min / amp max / fade
[*][b]mix[/b] -- dry/wet. Shift: input / output / tanh sat
[/list]

[b]New unit: Blanda[/b]
Three-input scan mixer (spreadsheet pkg). Pure mixer -- no onboard filtering. Each input has a bell-shaped response along a global Scan axis; sweeping Scan crossfades the three. Continuous polyline bell landscape with per-segment shading and territory-based ghost occlusion.
[list]
[*][b]in1 / in2 / in3[/b] -- per-input level + Solo/Mute. Expansion: Level / Weight (width) / Offset (position)
[*][b]scan[/b] -- global Scan position. Expansion: Skew (axis warp)
[*][b]focus[/b] -- bipolar bell-width scaler. Expansion: per-input Shape
[*][b]lvl[/b] -- output level
[/list]

[b]Helicase[/b] -- anti-alias shaping (polyBLEP on value jumps, smoothed-abs on V-corners), full 2x oversampling of the hi-fi inner loop with halfband decimation. 20 Hz DC blocker (bypassed below 1 Hz for LFO use).

[b]Petrichor[/b] -- grain system rebuilt: Hann envelope, 50% overlap, anchored spawn at tap reads, max grain 300 ms, stack cap 8. Parallel prefetch of all tap reads. Time fader throw restricted to buffer-size menu setting.

[b]Pecto[/b] -- three-pass tap loop (compute/gather/combine), 8-ahead prefetch, explicit NEON on passes A & C. Stereo @ density 24: ~50% -> ~6% CPU. 20 Hz DC blocker. Bipolar feedback.

[b]Other[/b] -- spreadsheet dual-mode GainBias controls guarded against nil focusedReadout (crash fix on mode transitions).
