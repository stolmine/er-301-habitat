[b]stolmine firmware + habitat v2.1.0[/b]

Focused polish pass on the spreadsheet package. 17 commits since v2.0.1.

[b]Larets[/b] -- stepped multi-effect overhaul. New CPR single-band compressor behind a single knob (Giannoulis/Massberg/Reiss feedforward, adapted from Impasto), auto-makeup toggle on a Parfait-style Mix ply. Grain-based pitch shift (two-grain Dattorro overlap, sin^2 windows). Hard-clip distortion with 1-20 drive and 1/drive^0.7 makeup, replacing the old tanh soft-knee -- distortion step now reads as a true limiter. Clock-locked stutter (musical-fraction beat repeat from a step-start buffer snapshot) and fresh-per-loop shuffle (new random source offset each repeat, within a two-tick window). Bitcrush direction inverted with a 12-bit ceiling. New bipolar (-1..1) global param offset ply between overview and xform -- non-destructive, applied per-sample so CV modulation tracks smoothly. Loop length reworked to 1-16 integer with a minimum of 1 for momentary single-step hold. Step-list type readout unified so the displayed abbreviation always matches what's stored. Tape stop and gate effects removed; the slots didn't have enough temporal headroom to do either idea justice in a stepped context. New reverse sweep, bitcrush particle fill, and concentric comb contour visualizations.

[b]Helicase[/b] -- cosmetic and expansion polish. Overview phase-space viz brightness now driven by a single measure (mean |dx| / RMS of the decimated output ring, approximately pitch-independent because the ring captures ~8 carrier cycles regardless of f0). Rest-state floor raised and blob gray ceiling lifted to white so peak FM pushes the viz all the way up. Sync phase-receptivity threshold now exposed on an expansion GainBias fader with a CV input -- sync ply main view is untouched.

Only spreadsheet bumped (2.0.0 -> 2.1.0). Other packages unchanged.

Delete the installed spreadsheet lib from the rear SD card to force reinstall:
[code]
sudo rm -rf /mnt/v0.7/libs/spreadsheet/
[/code]

Requires er-301-stolmine firmware. Compatible with vanilla community packages (Accents, sloop, lojik, strike, polygon).
